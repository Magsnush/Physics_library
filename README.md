This is a library for computing different cross sections at small x numerically. Currently it can compute DIS cross sections at LO using the optical theorem or using a finite energy constraint. 

In the building blocks folder I have implemented lightcone wavefunctions and multipole correlators in the MV-model.

In the scattering processes folders the integrands for computing cross sections exist based on the paper https://arxiv.org/abs/2601.07302.

In the numerics folders there exist example scripts of how to integrate the integrands in the scattering processes folder.
